## 重要限制

这个验证环境是 Windows 虚拟环境，只能验证 Python 依赖和 Windows 打包链路。它不能生成或验证 macOS 原生 `OA_Review.app`。

macOS `.app` 必须在 Mac 上运行：

```bash
cd /path/to/Astronaut_OA
bash build_macos.sh A1689
open A1689/OA_Review.app
```

Python 虚拟环境不能跨系统搬运，`.venv-win-verify` 不能复制到 macOS 使用。Mac 上需要重新创建 `.venv-oa-review`。
